Spotilist

Create a spotify playlist from a BBC Sounds episode.

This tool takes a BBC Sounds episode, extracts the songs, and puts them in a new Spotify playlist for you.

Instructions
  1. Enter the url of the BBC Sounds episode you want into the box below, which will go green if it's a valid URL.
  2. Click go and authenticate your Spotify account.
  3. Wait for a few seconds to be redirected to your new playlist.
Sometimes the tool struggles to find the BBC Sounds Episode that you want. If this happens try clicking "Go!" again.

A free and open source project by Ari Cooper-Davis, using CORS Anywhere and jQuery.
