Spotilist

Create a spotify playlist from a BBC Sounds episode.

This tool takes a BBC Sounds episode, extracts the songs, and puts them in a new Spotify playlist for you.

Instructions & More Options

Instructions:

  1. Enter the url of the BBC Sounds episode you want into the box above, which will go green if it's a valid URL.
  2. [optional] Fill in the desired playlist title and description below.
  3. Click go and authenticate your Spotify account.
  4. Wait for a few seconds to be redirected to your new playlist.
Sometimes the tool struggles to find the BBC Sounds Episode that you want. If this happens try clicking "Go!" again.
