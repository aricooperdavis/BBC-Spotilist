Spotilist

Create a spotify playlist from a BBC Sounds episode.

This site makes a spotify playlist containing all the tracks in a given BBC Sounds episode. All you need to do is:

  • Copy the url of the BBC Sounds episode you're interested in.
  • Paste the url into the box below.
  • Click go and let the tool do its thing. You'll need to authenticate your spotify account so that it can make you a playlist!
All of this is done in your browser using JavaScript, which maximises your privacy and security. As such you'll need a modern browser to use this tool. Please note that this tool is not affiliated with either the BBC or Spotify.


A free and open source project by Ari Cooper-Davis, using CORS Anywhere and jQuery.
